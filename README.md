## Ваш Каталог Групп (vk mini app)
## Зависимости
```shell
npm install --force
```

## Запуск
```shell
npm run dev
```

## Сборка
```shell
npm run build
```

## Публикация
Укажите `app_id` в файле [vk-hosting-config.json](./vk-hosting-config.json).
После чего выполните команду:
```shell
npm run deploy
```
